# Adventure Works - Sales Analysis
> 🚧 Work in Progress

## Overview
Power BI project analyzing reseller sales data
using Adventure Works DW 2020 database.
Built as part of a structured Power BI training program.

## Data Sources
- SQL Server: Adventure Works DW 2020
- ColorFormats.csv
- ResellerSalesTargets.csv

## Tables
- Sales (Fact Table)
- Product, Reseller, Region (Dimensions)
- Salesperson, SalespersonRegion, Targets

## What's Been Done So Far
- Data preparation and cleaning
- Data transformation and loading (Power Query)
- Data modeling: relationships, hierarchies,
  column formatting, data categories
  and hidden key columns

## In Progress
- DAX calculations
- Report building
- Dashboard creation
